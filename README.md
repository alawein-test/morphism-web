# Morphism Systems Website

Landing page for Morphism Systems - Enterprise Orchestration with Mathematical Guarantees.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Deployment

Deployed on Vercel at [morphism.systems](https://morphism.systems)

## Structure

- `pages/` - Page components (index.js is homepage)
- `pages/api/` - API endpoints (contact form)
- `styles/` - Global styles
- `public/` - Static assets

## Customization

### Colors
Edit `styles/globals.css` to change:
- Primary Blue: #003D82
- Secondary Orange: #FF6B35

### Content
Edit `pages/index.js` to update:
- Headlines
- Problem/solution cards
- Features
- CTA text
